Abstract

"Plinian Core is a standard data model designed to share biological species level information. It is developped as an XML schema. The **PlinianCore ontology** is a attempt to represent the XSD PlinianCore data model as an [OWL](https://www.w3.org/TR/2012/REC-owl2-overview-20121211/) ontology, to be used in RDF-based knowledge graphs."
